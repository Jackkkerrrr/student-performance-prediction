# Student Performance Prediction

This project explores several machine learning methods for predicting student performance on educational assessment data.

## Methods Implemented

- Majority Vote Baseline
- K-Nearest Neighbors (KNN)
- Item Response Theory (IRT)
- Matrix Factorization
- Neural Network
- Ensemble Learning

## Dataset

The project uses student-question response data and metadata provided in the course assignment.

Files include:

- train_data.csv
- valid_data.csv
- test_data.csv
- question_meta.csv
- student_meta.csv
- subject_meta.csv

## Results

The performance of different models was evaluated on the validation and test datasets.
